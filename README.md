# ANIMATIONS

Add link in head
```
<link rel="stylesheet" href="https://unpkg.com/aos@next/dist/aos.css" />
```
Add link at the end before closing body tag, and initialise AOS

```
<script src="https://unpkg.com/aos@next/dist/aos.js"></script>
<script>
    AOS.init();
</script>
```
